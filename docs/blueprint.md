# **App Name**: EdgeCipher

## Core Features:

- AppShell Layout: Create a responsive layout with a fixed left sidebar and a main content area with a top header.
- Navigation: Implement routing for all pages, each displaying a placeholder view.
- Reusable Components: Implement 'Card', 'Button' (primary, secondary, ghost), 'MetricTile', and 'Tag' components.
- Tooltip Component: Create a reusable tooltip component that displays helper text on hover/click.
- Performance Insights: Utilize an AI tool to generate tailored insights on a user's past trading performance, risk tendencies, and habits.
- Trade Planning Tool: AI-powered recommendations on ideal strategies tailored to user and historical data.
- Mocked Data: Use mocked data for all views to simulate real-time trading information.

## Style Guidelines:

- Background: Very dark navy (#0A192F) to provide a sophisticated base.
- Surfaces: Slightly lighter darks with subtle gradients to create depth. (#172A45)
- Primary: Bright electric blue (#64FFDA) for focus and branding.
- Accent/Warning: Red (#FF4500) for critical alerts and actions.
- Text: Mostly white (#FFFFFF) with softer secondary text for readability and contrast.
- Body and headline font: 'Inter' sans-serif for a modern, machined feel. Note: currently only Google Fonts are supported.
- Desktop-first, responsive design to ensure a seamless experience across devices.
- Use minimalistic, vector-based icons for a clean and modern look.
